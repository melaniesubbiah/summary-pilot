### Task Description
For each question you will see two passages of AI-generated text, either summaries or emails, to compare based on a set of style directions. You will have to answer which passage better follows the style direction. You can ignore the content of the passages and focus on the writing style. If neither passage follows the style direction, you can select neither.

Use the 'Next' button to navigate to the next question.
